韓語方塊字組合邏輯 (全6種型態)

韓文是拼音文字，每一個「方塊字」至少需要一個子音和一個母音構成，最多可以包含兩個尾音（收音）。

1. 左右結構 (初聲 + 垂直母音)

規則：子音在左，垂直型母音(ㅏ,ㅑ,ㅓ,ㅕ,ㅣ)在右。

範例：ㄱ + ㅏ = 가 (ga)

2. 上下結構 (初聲 + 水平母音)

規則：子音在上，水平型母音(ㅗ,ㅛ,ㅜ,ㅠ,ㅡ)在下。

範例：ㄱ + ㅗ = 고 (go)

3. 複合結構 (初聲 + 複合母音)

規則：子音在左上，複合母音(ㅘ,ㅝ,ㅢ等)包圍在下與右。

範例：ㄱ + ㅘ = 과 (gwa)

4. 左右 + 底部收音 (初聲 + 垂直母音 + 終聲)

規則：左右結構的下方加上一個子音作為收音。

範例：ㄱ + ㅏ + ㄴ = 간 (gan)

5. 上下 + 底部收音 (初聲 + 水平母音 + 終聲)

規則：上下結構的下方加上一個子音作為收音。

範例：ㄱ + ㅗ + ㅁ = 곰 (gom)

6. 複合 + 底部雙收音 (極限狀態)

規則：複合母音結構下方加上雙子音收音。

範例：ㄱ + ㅙ + ㄴ = 괜 (gwaen) (如 괜찮다)

基本句型結構與語序 (全解析)

韓語是黏著語，依靠「助詞」來標記單詞在句子中的成分。其核心語序為 SOV (主語 + 賓語 + 動詞)。

1. 名詞述語句 (A是B)

語序韓文範例中文翻譯
主語 + 名詞 + 이다(是)저는 학생입니다.我是學生。
主語 + 名詞 + 가/이 아니다(不是)저는 의사가 아닙니다.我不是醫生。

2. 形容詞述語句 (主語的狀態)

語序韓文範例中文翻譯
主語 + 形容詞날씨가 좋습니다.天氣很好。
主語 + 程度副詞 + 形容詞한국어가 아주 재미있습니다.韓文非常有趣。

3. 動詞述語句 (完整 SOV 結構)

語序韓文範例中文翻譯
主語 + 動詞비가 옵니다.下雨了。
主語 + 賓語 + 動詞저는 사과를 먹습니다.我吃蘋果。

4. 包含時間與地點的完整語序

標準排序：主語 + 時間 + 地點 + 賓語 + 動詞

語序拆解韓文範例對應中文
主語 (S)저는 (我)我
時間 (T)오늘 (今天)今天
地點 (P)도서관에서 (在圖書館)在圖書館
賓語 (O)한국어를 (韓文)韓文
動詞 (V)공부합니다. (學習)學習。
完整句子：저는 오늘 도서관에서 한국어를 공부합니다. (我今天在圖書館學習韓文。)

5. 否定句語序 (兩種形式)

否定類型語序韓文範例中文翻譯
短形否定主語 + 賓語 + 안 + 動詞저는 고기를 안 먹어요.我不吃肉。
長形否定主語 + 賓語 + 動詞詞幹 + 지 않다저는 고기를 먹지 않아요.我不吃肉。
